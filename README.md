# ❄️ Winter Gallery

A beautiful photography gallery website built with HTML5 and CSS3.

## 📌 About

Winter Gallery is a responsive landing page that showcases stunning winter photography. It features a clean and minimal design with a light blue/grey color palette inspired by the winter season.

## ✨ Features

- Responsive Navbar with Snowflake Logo
- Full Screen Hero Section
- Filterable Photo Gallery (All, Landscapes, Decorations, Food, Family)
- Masonry Style Grid Layout
- Hover Effects with Title and Category
- About Section
- Contact Form
- Clean Footer

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts (Cormorant Garamond, Montserrat)
